# Проект: «Дневник тренировок»

## 1. О проекте
**Название:** Дневник тренировок  
**Цель:** Создание приложения для ведения персональных записей о физических занятиях.  
Система должна позволять пользователю сохранять и анализировать свои тренировки, отслеживать прогресс, планировать будущие занятия и фиксировать достижения.

---

## 2. Описание системы
Приложение ориентировано на людей, занимающихся спортом самостоятельно или под руководством тренера.  
С его помощью можно структурировать тренировочный процесс и хранить информацию о каждом занятии.

Основные задачи системы:
- ведение списка тренировок по датам;
- фиксация упражнений, количества подходов и повторений;
- возможность добавления комментариев и заметок;
- отслеживание прогресса и достижений пользователя.

---

## 3. Основные возможности

###  Авторизация и регистрация
- Создание нового профиля с логином и паролем;  
- Вход в систему под своей учетной записью;  
- Хранение пользовательских данных в базе.

###  Управление тренировками
- Добавление новой тренировки с указанием даты и типа (силовая, кардио и т.д);  
- Добавление упражнений к каждой тренировке (название, количество повторений, вес, время);  
- Возможность редактирования и удаления тренировок;  
- Поиск по дате или названию упражнения.

###  Статистика 
- Просмотр графиков прогресса (например, рост рабочего веса или количества повторений);  
- Подсчёт общего количества тренировок за период.

###  Заметки и комментарии
- Добавление кратких комментариев к каждой тренировке;  
- Ведение личных заметок.

---

## 4. Роли пользователей
| Роль | Описание возможностей |
|------|------------------------|
| **Обычный пользователь** | Ведение своего дневника, добавление и редактирование тренировок, заметок и аналитики |
| **Тренер / администратор** | Просмотр дневников клиентов, добавление рекомендаций, контроль и анализ данных |

---

## 5. Интерфейс приложения
**Основные разделы:**
1. Экран входа / регистрации  
2. Главная страница с краткой сводкой последних тренировок  
3. Раздел «Мои тренировки» — список занятий с фильтрацией  
4. Раздел «Аналитика» — визуализация прогресса  
5. Раздел «Заметки» — личные комментарии  
6. Панель администратора (для тренера)

---

## 6. Техническая реализация
- **Язык программирования:** Java 
- **Хранение данных**
- **Структура:** объектно-ориентированная (модули для работы с пользователями, тренировками и аналитикой)  
- **Тестирование:** модульные тесты для проверки корректности добавления, редактирования и анализа данных.
